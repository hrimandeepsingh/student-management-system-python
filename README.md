# student-management-system-python
A simple menu-driven Student Management System built using Python basics.

This is a simple menu-driven Student Management System built using Python.

## Features
- Add student records
- View all students
- Search student by roll number
- Delete student record

## Concepts Used
- Python basics
- Lists and dictionaries
- Functions
- Conditional statements
- Loops

This project was created as part of learning Python fundamentals.
